# ResQNet Mobile App

React Native mobile application for emergency response on iOS and Android.

## Coming Soon

### For Patients
- Emergency SOS button
- Location sharing
- Real-time ambulance tracking
- Emergency history

### For Paramedics
- Emergency notifications
- Turn-by-turn navigation
- Patient information access
- Status updates

### For Hospital Staff
- Incoming patient alerts
- Bed capacity management
- ETA tracking

## Tech Stack (Planned)

- React Native with TypeScript
- Expo for cross-platform development
- Azure Maps SDK for navigation
- Push notifications via Azure Notification Hubs
- Redux Toolkit for state management

## Setup

```bash
npm install
npx expo start
```
