# ResQNet Frontend

React-based web application for emergency response dispatch and monitoring.

## Coming Soon

- **Dashboard**: Real-time emergency monitoring
- **Dispatch Console**: Ambulance and hospital assignment
- **Map View**: Live tracking of ambulances
- **Patient Portal**: Emergency reporting interface
- **Analytics**: Emergency response metrics

## Tech Stack (Planned)

- React 18+ with TypeScript
- Azure Maps integration
- Socket.IO for real-time updates
- Tailwind CSS for styling
- React Query for state management

## Setup

```bash
npm install
npm run dev
```
